Chemistry Safety Assistant

Safety is my passion™, so I built Chemistry Safety Assistant to make it quick and easy to get safety information about chemicals. Users can draw chemical structures or search by name or SMILES, and recieve back a table of hazard information sourced from PubChem.

The back end for Chemistry Safety Assistant is written in Java, and the front end is in JavaScript using the React framework. It was my capstone project for Ada Developer's Academy.

Task List API

It sometimes seems like there's never enough hours in the day, so Adies get organized with Task List API. Storing their to-dos in a postgres database, they write RESTful CRUD routes to sort and organize that pesky task backlog. To keep up morale, Task List calls the Slack API to post congratulations upon task completion.
